Chat-Rmi

Um simples chat via Terminal usando RMI

View on GitHub Download .zip Download .tar.gz

Chat Rmi para projeto de Sistemas Distribuidos.

Alunos: Tales Marinho Godois , Vinicius de Avelar Oliveira , Vinicius Pacheco Gomes

O programa consiste em dois pacotes principais, o Server e o Client.

Para o chat entrar em funcionamento, primeiro, é necessário executar, pela IDE escolhida, a classe Exec do pacote Server. Em seguida é necessário executar a classe Exec dos clientes( Cada um representa um usuario). Os Usuarios devem inserir seu nome e teclar enter, será informado se o usu;ario foi criado, ou se o nome já consta no chat.

Em seguida para enviar mensagens basta o usuario escrever o que deseja enviar e isso será enviado para todos do grupo ao se teclar Enter.
